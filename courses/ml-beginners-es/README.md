## Contenido por Parte

### Parte 1: Fundamentos
- **Introducción al ML:** Qué es ML, historia, fairness, técnicas
- **Regresión:** Herramientas, datos, regresión lineal, regresión logística

### Parte 2: Clasificación
- **Web App:** Construcción de aplicaciones web con ML
- **Clasificación:** Introducción, clasificadores, aplicaciones
- **Clustering:** Visualización, K-Means

### Parte 3: NLP
- **Procesamiento de Lenguaje Natural:** Introducción, tareas, traducción, análisis de sentimiento, reseñas de hoteles

### Parte 4: Avanzado
- **Series de Tiempo:** Introducción, ARIMA, SVR
- **Aprendizaje por Refuerzo:** Q-Learning, OpenAI Gym
- **Aplicaciones del Mundo Real:** Aplicaciones, debugging de modelos ML
